# **App Name**: Movilo.club

## Core Features:

- Plan Display: Clearly present the different health discount plans (Individual, Family, Pareja) with their respective prices and benefits.
- Provider Search: Enable users to search for affiliated clinics, labs, specialists, and pharmacies by zone and/or specialty. Bilingual support included.
- Interactive Map: Integrate an interactive map showing the locations of affiliated healthcare providers in Cali. Bilingual support included.
- FAQ Section: Provide a comprehensive FAQ section to answer common questions about coverage, plan differences, and membership details. Bilingual support included.
- User Authentication: Secure user accounts and session management, tied to Supabase Auth and SvelteKit hooks for validation and route protection.
- Digital Card Download: Allow logged-in users to download a digital version of their discount card with a QR code for easy access.
- Contact Form: Implement a contact form and provide contact information (phone, WhatsApp, email, chat) for customer support. Bilingual support included.
- Client Dashboard: Dashboard for clients to see benefits, search for services, make appointments, rebuy affiliations, print cards, and see benefits usage per person.
- Provider Dashboard: Dashboard for providers to offer their services, input data (locations, addresses, products), see their customers, and access a basic CRM with client demographics and history.
- Admin Dashboard: Dashboard for Movilo.club admin to see data from both clients and providers (affiliations per day, client satisfaction surveys), and accept providers.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to evoke trust and health.
- Background color: Light gray (#F0F4F7) for a clean and modern look.
- Accent color: A warm green (#81C784) to highlight key benefits and CTAs.
- Body and headline font: 'Inter' for a modern, neutral, and readable style across all content.
- Use simple and recognizable icons to represent the benefits of each plan and the types of healthcare providers.
- Use a clear, modular layout with plenty of white space to ensure readability and ease of navigation.
- Subtle animations for page transitions and loading states to enhance the user experience.
